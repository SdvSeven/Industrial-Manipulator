## О проекте

Программа позволяет управлять промышленным манипулятором в реальном времени с помощью жестов ладони, распознаваемых через камеру библиотекой **MediaPipe**. Команды передаются на аппаратную часть через **Arduino** по последовательному порту.

Проект ориентирован прежде всего на применение в **химической промышленности** — там, где оператор должен управлять манипулятором дистанционно, не контактируя с опасными веществами.

> ⚠️ Проект находится в активной разработке с **01 октября 2025 года**.

## Возможности

| Функция | Описание |
|---|---|
| 👋 Жестовое управление | Распознавание жестов руки через MediaPipe |
| 🕹️ Ручное управление | Кнопки-стрелки как резервный режим (fallback) |
| 📡 Подключение устройств | Arduino по COM-порту, выбор камеры по индексу |
| ⛔ Emergency Stop | Мгновенная остановка всех сервоприводов |
| 📋 Журнал событий | Лог всех жестов и команд с временными метками |
| ⚙️ Гибкая настройка | Маппинг жестов, скорость, чувствительность, разрешение |

## Страницы приложения

### 🏠 Home — Центр управления подключением
Стартовая страница. Отображает статус всех устройств и позволяет запустить систему одной кнопкой.

```
🟢 Arduino (COM3)   🟢 Camera (0)   🟡 MediaPipe   🔴 Connection
```

### 🎮 Control — Управление
Основная рабочая страница. Слева — видеопоток с overlay жеста и команды, справа — панель управления с режимами, слайдерами и аварийной остановкой.

### ⚙️ Settings — Настройки
Конфигурация Arduino, камеры и таблицы сопоставления жестов с командами.

### 📋 Logs — Журнал
Таблица всех событий с фильтрами по времени и типу команды, экспорт в CSV.

## Жесты и команды

| Жест | Команда |
|---|---|
| LEFT | MOVE_X -10 |
| RIGHT | MOVE_X +10 |
| UP | MOVE_Y +10 |
| DOWN | MOVE_Y -10 |
| FIST | GRAB |
| OPEN | RELEASE |

> Таблица маппинга редактируется в разделе **Settings**.

## Требования

- Python 3.10+
- Arduino с прошивкой для управления сервоприводами
- Веб-камера (индекс 0, 1 или 2)
- Зависимости: `mediapipe`, `opencv-python`, `pyserial`, `PyQt5` (или аналог)

## Установка и запуск

```bash
# Клонировать репозиторий
git clone <repo-url>
cd <repo-folder>

# Установить зависимости
pip install -r requirements.txt

# Запустить приложение
python main.py
```

## Состояния системы

| Состояние | Описание |
|---|---|
| `IDLE` | Система ожидает запуска |
| `CONNECTING` | Идёт подключение устройств |
| `RUNNING` | Система работает в штатном режиме |
| `ERROR` | Обнаружена ошибка, работа невозможна |
